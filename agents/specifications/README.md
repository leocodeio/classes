## Quick Start

### 1. Create Content

```bash
# Create a new markdown file
touch content/webdev/css_basics.md
```

### 2. Use Template

Copy the template from [markdown-template.md](./markdown-template.md) and customize.

### 3. Generate JSON

```bash
pnpm agent:slides content/webdev/css_basics.md
```

### 4. View Slides

Navigate to `/slides/webdev/css-basics` in the web app.
